WildRats Signature Generator

WTF is WRSG?

WildRats Signature Generator is a service that was cooked to help other WILDSTAR players to generate a simple, concise but stylish signature... and... admit it, it's just cool to generate your fucking WILDSTAR signature in a minute!

You just have to specify a value for each different parameters and this service will automatically generate your signature.

Here is the kind of address to use => http://localhost/signature/wrs/[faction]/[guild_name]/[race]/[name]/[class]/[role]/[path]/[server_name]/[effect]/[size]

Find below a description of the different parameters and an example (no excuses naughty boy) :

  • param1 FACTION : exiles|dominion
  • param2 GUILD NAME : your guilde name, any string
  • param3 RACE : human|granok|orin|mordesh|cassian|mechari|draken|chua
  • param4 NAME : your character name, any string
  • param5 CLASS: warrior|esper|spellslinger|engineer|stalker|medic
  • param6 ROLE : melee_dps|ranged_dps|tank|healer
  • param7 PATH : soldier|explorer|settler|scientist
  • param8 SERVER NAME : the server name your are playing on, any string
  • param9 EFFECT : depends if you want a bloody effect on your signature => no_effect|gore|neon|drug
  • param10 SIZE : signature size => normal|small
